# Finance Tracker (Python, SQLite, CustomTkinter)

A simple desktop finance tracking app built with **Python**, **SQLite**, and **CustomTkinter**.  
It lets you:

- Track income and expenses
- Categorize transactions (Work, Allowance, Food, School, etc.)
- Store data in a local SQLite database
- See your running balance update in real time
